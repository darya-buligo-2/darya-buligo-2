3. Общая структура рекламного агентства

Рекламное агентство состоит из нескольких функциональных подразделений, каждому из которых вменяются определенные функции.

Руководитель рекламного агентства координирует всю работу, отвечает за результаты деятельности, претворяет в жизнь задумки руководства предприятия.

Рабочая группа по разработке рекламы непосредственно подотчетна руководителю рекламного агентства, занимается разработкой новых технологий по производству рекламы, отвечает за производственную  и научную стороны деятельности рекламного агентства.

Рабочая группа по PR отвечает за связи с общественностью, защиту деловой репутации клиентов и т.д.

Специалист по личным продажам руководит организацией различных встреч, промо-акций, разрабатывает стратегию личных продаж.

Рабочая группа по маркетинговым исследованиям занимается постоянным изучением рынка, проведением маркетинговых исследований.
